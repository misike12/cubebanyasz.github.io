Fix App/Certificate not trusted error

Download Certificate

Steps

1) Download Certificate file 
2) Open File  

3) Click on Install Certificate

4) Select Local Machine and click Next

5) Select "Place all certificates in following store" and then click on Browse

6) Select "Trusted Root Certification Authorities" and click Ok

7) Click Next

8) Click Finish
